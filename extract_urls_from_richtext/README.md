# セル内のリッチテキストからURLを自動抽出

スプレッドシートのセル内のテキストに埋め込まれたリンク（リッチテキスト形式）から、URLのみを抽出して自動で書き出すスクリプトです。

## 仕様

* **入力元の列:** 現在アクティブなシートの `A列`（1行目から最終行までを自動判定）。
* **出力先の列:** `B列` 以降。
* **複数URLの処理:** A列の1つのセル内に複数のリンクが含まれている場合、B列、C列、D列…と横方向に分割して出力されます。重複するURLは除外されます。
* **注意点:** 実行時、出力先となる範囲のデータは一度クリアされます。B列以降に消えては困るデータがある場合は、事前に退避させるか空の列を用意してください。

## 導入と使い方

1. 対象のスプレッドシートを開き、メニューの **「拡張機能」** > **「Apps Script」** をクリックします。
2. 開いたエディタに、本ディレクトリの `code.gs` の内容をコピーして貼り付け、保存します。
3. スプレッドシートのA列に、リンクが埋め込まれたテキストを用意します。
4. エディタ画面に戻り、上部の **「▷ 実行」** ボタンをクリックします。
   * ※初回実行時のみ、Googleアカウントへのアクセス承認画面が表示されます。「詳細」から安全ではないページへ移動し、許可を行ってください。
5. 実行完了後、スプレッドシートのB列以降に抽出されたURLが出力されます。
